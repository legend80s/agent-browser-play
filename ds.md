## 简单介绍嘉兴红船

**嘉兴红船**是位于浙江嘉兴南湖上的一艘**纪念船**，象征着**中国共产党的诞生地**，是重要的红色革命象征。

## 历史背景

### 中共一大与红船

- **1921年7月**：中共一大在上海法租界秘密召开
- **被迫转移**：会议遭法租界巡捕袭扰，被迫中断
- **转战南湖**：代表们转移到嘉兴南湖，在一艘游船上继续会议
- **历史时刻**：1921年8月初，会议在南湖游船上闭幕，宣告中国共产党正式成立

## 红船本体

| 特征     | 说明                       |
| -------- | -------------------------- |
| **船型** | 单夹弄丝网船（江南画舫）   |
| **尺寸** | 长约16米，宽约3米          |
| **构造** | 前舱、中舱、房舱、后舱     |
| **外观** | 雕梁画栋，典型江南游船风格 |
| **位置** | 南湖湖心岛岸边             |

## 红船的象征意义

- **诞生地象征**：代表中国共产党诞生的历史时刻
- **精神象征**："红船精神"——开天辟地、敢为人先的首创精神
- **初心象征**：承载着共产党人的初心和使命

## 红船精神内涵

2005年，习近平同志首次提出"红船精神"，核心内涵：

| 内涵         | 说明               |
| ------------ | ------------------ |
| **首创精神** | 开天辟地、敢为人先 |
| **奋斗精神** | 坚定理想、百折不挠 |
| **奉献精神** | 立党为公、忠诚为民 |

## 历史意义

红船不仅是：

- **物理存在**：一艘停泊在南湖的纪念船
- **历史见证**：中国共产党诞生的见证者
- **精神图腾**：中国共产党人的精神家园
- **教育载体**：开展爱国主义教育的重要场所

## 相关景点

- **南湖革命纪念馆**：展示建党历史，馆藏一大史料
- **湖心岛烟雨楼**：登楼俯瞰红船全景
- **七一广场**：大型纪念活动场所

简单说：**嘉兴红船 = 中共诞生地 + 红船精神发源地，是停泊在南湖上的革命象征，代表中国共产党启航的地方**。
